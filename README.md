# סטודיו דניאל — אתר עיצוב פנים

אתר סטטי (`index.html` + `about.html` + `portfolio.html` + `contact.html`)
לעסק עיצוב פנים. עברית RTL, פלטה בהירה-חמימה, ללא build, ללא framework.

🔗 **חי באוויר:** https://liamhemo64.github.io/daniel-design/

---

## מה יש עכשיו

- **index.html** — Hero (טקסט + תמונת אווירה) → גלריית "לפני | אחרי" סטטית
  לכל חדר (מטבח, סלון, חדר שינה, חדר רחצה) → פרויקטים נבחרים → ציטוט
  לקוחה → טיזר אודות → CTA/WhatsApp.
- **about.html** — סיפור הסטודיו + תהליך עבודה ב-4 שלבים.
- **portfolio.html** — גריד פרויקטים.
- **contact.html** — וואטסאפ / טלפון / אינסטגרם + טופס שפותח וואטסאפ (ללא שרת).
- נגישות: `prefers-reduced-motion` מכובד, ניגודיות טקסט, focus states.
