# סטודיו דניאל — הרחבה לאתר רב-עמודי + רידיזיין בהיר-חמים

תאריך: 2026-07-06 · סטטוס: מאושר (גישה A)

## מטרה

להפוך דף נחיתה בודד (scrollytelling כהה) לאתר סטודיו מלא: 4 עמודים,
פלטה בהירה-חמימה ("חם ונעים אבל אלגנטי"), עברית RTL, ללא build.

## החלטות

- **גישה A**: כל האתר בפלטה בהירה-חמימה אחת. מנוע ה-scrollytelling נשאר כמו שהוא.
- **מבנה**: קבצים סטטיים נפרדים, tokens משוכפלים בכל קובץ (בכוונה — אפס build).
- **תוכן אמיתי**: אין עדיין. הכל placeholders מסומנים `TODO: SWAP PLACEHOLDER`.

## עמודים

### 1. `index.html` (רידיזיין)
- Hero בהיר-חמים → 4 פרקי scrollytelling (מנוע ללא שינוי, כולל תמיכת image
  sequence) → רצועת טיזר "אודות" (משפט + לינק ל-about) → CTA וואטסאפ → footer.
- בפרקים עצמם (stage) התמונות נשארות עם scrim כהה לקריאוּת — זה תקין ומכוון:
  קונטרסט דרמטי בתוך במה, בהיר סביבה.

### 2. `about.html`
- פורטרט placeholder, טקסט סיפור placeholder.
- תהליך עבודה: פגישה → קונספט → ביצוע → מסירה (4 צעדים ממוספרים).

### 3. `portfolio.html`
- גריד פרויקטים: קלף = תמונת שער (Unsplash placeholder), שם פרויקט, תגיות
  סוג-חדר. ללא lightbox. 6 קלפים לדוגמה.

### 4. `contact.html`
- כפתור וואטסאפ, לינק `tel:`, לינק אינסטגרם, טופס פשוט שפותח וואטסאפ עם
  טקסט ממולא (ללא backend).

## משותף לכל העמודים

- Nav עליון דק, קבוע (fixed), RTL: לוגו-טקסט מימין, לינקים משמאל,
  עמוד נוכחי מסומן בפליז. במובייל — שורת לינקים דחוסה (ללא המבורגר; 4 לינקים
  קצרים נכנסים ב-375px).
- Footer אחיד: שם עסק, טלפון placeholder, אינסטגרם, שנה.
- נגישות: focus-visible, `prefers-reduced-motion` (קיים ב-index), alt לתמונות.
- Mobile-first 375px, הרחבות ב-768/1000px — כמו הקיים.
